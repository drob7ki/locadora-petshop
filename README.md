**PetLoc 🐾**

Sistema web desenvolvido em PHP para gerenciamento de serviços de um petshop, permitindo controle de reservas, hospedagens e serviços de banho para pets.

O projeto utiliza Programação Orientada a Objetos (POO), autenticação de usuários, armazenamento em JSON e interface responsiva com Bootstrap.

**📌 Funcionalidades**
* Login de usuários

* Controle de permissões (Administrador e Usuário)

* Cadastro de serviços pet

* Reserva de serviços

* Finalização de serviços

* Exclusão de serviços

* Cálculo de previsão de valores

* Persistência de dados em arquivos JSON

* Interface responsiva e moderna
